# Tienda Digital en Java
Este proyecto es una simulación de una tienda digital desarrollada en Java como parte de una actividad académica.

## Objetivo
Aplicar conceptos de Programación Orientada a Objetos (POO) y el flujo de trabajo Git Flow.

## Funcionalidades
- Gestión de productos
- Registro de clientes
- Simulación de compras
- Carrito de compras
- Reporte básico de tienda

## Estructura del proyecto
- Producto.java
- Cliente.java
- Carrito.java
- Tienda.java (opcional)
- Main.java

## Tecnologías usadas
- Java
- Git y GitHub

## Ejecución
1. Compilar: javac *.java
2. java Main
